POLY

A hobby game engine in Vulkan

GAME ENGINE

Poly is a vulkan engine that I work on during my freetime. The current features are quite sparse but they are being worked on a lot to make the final experience with the interface as impeccable as possible.

CURRENT FEATURES

  • Vulkan API usage
  • Graphics API abstraction
  • SPIR-V online compiler using GLSlang

CURRENTLY WORKING ON

  • Render Graph
